03

GitHub에 배포하기

작성된 코드를 GitHub Pages를 통해 온라인에 공개하는 순서입니다.

1. GitHub 저장소 생성

GitHub 웹사이트에서 새로운 Repository를 생성합니다 (Public 설정).

2. 로컬 프로젝트 초기화 및 커밋

VS Code 터미널에서 다음 명령어를 순서대로 입력하세요.

git init
git add .
git commit -m "Initial commit"

3. 저장소 연결 및 푸시

생성한 GitHub 저장소 주소를 사용하여 연결하고 코드를 업로드합니다.

git remote add origin [당신의_레포지토리_URL]
git branch -M main
git push -u origin main

4. GitHub Pages 활성화

Repository Settings > Pages 메뉴로 이동합니다.
Build and deployment 소스를 Deploy from a branch로 설정하고,
main 브랜치 / (root) 폴더를 선택 후 Save를 클릭합니다.
