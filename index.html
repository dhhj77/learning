03

GitHub에 배포하기 (npm)

gh-pages 패키지를 사용하여 명령어 한 번으로 쉽게 배포하는 방법입니다.

1. 패키지 초기화 및 설치

터미널에서 npm init -y로 초기화 후, 배포 도구를 설치합니다.

npm install gh-pages --save-dev

2. package.json 설정

homepage 주소를 추가하고 deploy 스크립트를 작성합니다.

"homepage": "https://사용자명.github.io/저장소명",
"scripts": {
  "deploy": "gh-pages -d ."
}

3. 배포 실행

다음 명령어로 GitHub Pages에 즉시 배포합니다.

npm run deploy
